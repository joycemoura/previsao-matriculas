# 📊 Previsão de Matrículas Escolares

Aplicação desenvolvida em Python e Streamlit para prever o número de matrículas de escolas do estado do Paraná com base em características de infraestrutura e porte escolar.

## 🚀 Tecnologias Utilizadas

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-Learn
- Joblib
- Matplotlib

## 📁 Base de Dados

Os dados utilizados são provenientes do Censo Escolar de 2023, contendo informações sobre infraestrutura, quantidade de salas, professores e matrículas das escolas.

## 🤖 Modelo de Machine Learning

Foi utilizado um modelo de regressão treinado para estimar a quantidade de matrículas com base em variáveis relacionadas à infraestrutura escolar.

## 📈 Métricas de Avaliação

| Métrica | Valor |
|----------|----------|
| MAE | 84,56 |
| RMSE | 169,69 |
| R² | 0,68 |
